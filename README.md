## Project Overview

This project analyzes **public discourse on the Epstein Files on X (Twitter)** using scraped social media data, sentiment analysis, and NLP to uncover opinion shifts, narrative clusters, and emerging conversation trends.

Using an X scraping dataset and a Python-based analysis pipeline powered by **VADER** and text analytics, the project simulates how social listening and media intelligence teams can turn unstructured public conversations into actionable insights.

---

## Context, Problem, and Users

### 1. Context  
High-profile topics on social platforms generate fragmented and fast-moving narratives, making it difficult to track sentiment and dominant discussion themes.

### 2. Problem  
How can X discourse around the Epstein Files be analyzed to detect shifts in sentiment, identify narrative clusters, and surface emerging public opinion trends?

### 3. Users  
Media Intelligence, Social Listening, Research, and Public Affairs teams.
